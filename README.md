# Manutenção de Site - AI Services 🤖🛠️

Este repositório contém o exercício prático de manutenção e correção de código da disciplina de Desenvolvimento para Internet.

## 🎯 Sobre o Projeto (O Cenário)
A atividade simula um cenário real onde os arquivos do site de uma empresa de tecnologia fictícia (**AI Services**) foram corrompidos em produção. 

O objetivo do exercício foi realizar a engenharia reversa e a manutenção completa do código legado, atuando na correção estrutural e visual sem o uso de folhas de estilo externas.

## ⚙️ Manutenções Realizadas
Conforme o escopo da solicitação de manutenção, as seguintes correções foram aplicadas:
1. **Limpeza de Código:** Fechamento de tags HTML abertas, correção de erros de digitação e organização da indentação.
2. **Estrutura Semântica:** Inserção de tags `<header>`, `<main>` e `<footer>` para melhor organização da página.
3. **SEO e Metadados:** Inclusão das tags `<meta charset="UTF-8">`, `<meta name="viewport">` e `<meta name="description">` em todos os documentos.
4. **Padronização Visual (CSS Inline):** Aplicação de uma identidade visual consistente usando apenas o atributo `style`, padronizando fontes (Arial), cores de fundo e tamanhos de texto em todas as páginas.
5. **Menu de Navegação:** Criação de um menu universal padronizado para interligar todo o ecossistema do site de forma fluida.
6. **Desenvolvimento de Novas Funcionalidades:** Criação das páginas `planos.html` e `clientes.html` do zero, seguindo os critérios de aceite estabelecidos.
